What is Deepr Ai?

Deeper-ai is a beginner-friendly neural network library developed in Python and Cython. Deepr-ai is super simple and easy to use. Deepr-ai is made for a level 0 introduction to machine learning to get an understanding of the core concepts and to prepare the user for more advanced frame works.

Whats ​Different?

With Deepr-ai's super easy and simple setup it requires almost no previous knowledge in Python, math, and machine learning to set up a model. However there are more advanced setting for more experanced users.
